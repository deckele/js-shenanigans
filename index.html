Security issues found in NPM last year

eslint-scope@3.7 and eslint-config-eslint@5.0 (July)

  • Millions of users
  • Stole private user information
  • Attacker got access to the credentials of a maintainer
  • Attack through a postinstall script, which downloads and execs code
  • Script was found only because of script bug on slow connections!

event-stream@3.3.6 (Sep - November)

  • Maintainer gave control of the repo willingly to the attacker
  • shipped with the malicious dependency: flatmap-stream
  • Stole Bitcoin wallet information
